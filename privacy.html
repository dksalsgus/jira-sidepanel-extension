개인정보처리방침

최종 업데이트: 2025년 3월 18일

중요: Please Be Done 확장 프로그램은 사용자의 개인정보를 수집하거나 저장하지 않습니다.

1. 정보 수집

본 확장 프로그램은 다음 정보만 로컬 저장소에 저장합니다:

  • Jira 도메인 URL
  • Jira 사용자 이메일
  • Jira API Token

이 정보는 오직 사용자의 Jira 인스턴스에 연결하여 티켓을 표시하는 목적으로만 사용됩니다.

2. 데이터 저장

모든 데이터는 사용자의 로컬 브라우저 저장소(chrome.storage.sync)에만 저장됩니다.

  • 서버로 데이터 전송하지 않음
  • 제3자와 데이터 공유하지 않음
  • 개발자는 사용자의 데이터에 접근할 수 없음

3. 네트워크 요청

확장 프로그램은 다음 도메인으로만 요청을 보냅니다:

  • 사용자가 지정한 Jira 인스턴스 (예: *.atlassian.net)

4. 쿠키 및 추적

본 확장 프로그램은 쿠키를 사용하거나 사용자를 추적하지 않습니다.

5. 데이터 삭제

사용자는 언제든지 확장 프로그램 설정에서 저장된 인증 정보를 삭제할 수 있습니다.

확장 프로그램 제거 시 모든 데이터가 자동으로 삭제됩니다.

6. 변경 사항

본 개인정보처리방침은 개인정보 처리 방식이 변경될 경우 업데이트됩니다.

7. 문의

개인정보와 관련한 문의사항이 있으시면 GitHub 저장소의 이슈를 통해 문의해 주세요.
